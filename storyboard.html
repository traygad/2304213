Storyboard and Website Features

Website Layout

The website consists of six pages: Home, Professional, Personality, Personal Development, References, and Storyboard. Each page has a common navigation bar for easy navigation.

Wireframe

A hand-drawn wireframe for the layout of the website is attached below. It illustrates the structure and design of each page.

List of Features

  • Navigation bar for easy access to all pages.
  • Clickable image on the index page that opens an email client.
  • Internal links (bookmarks) to jump to sections on the same page.
  • Responsive design using basic CSS to improve the layout on different screen sizes.
